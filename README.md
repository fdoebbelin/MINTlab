# MINTlab – Interdisziplinäres, interaktives Lernen im Browser

[![Starte MINTlab im Browser](https://mybinder.org/badge_logo.svg)](https://mybinder.org/v2/gh/fdoebbelin/MINTlab/HEAD?labpath=notebooks/00_TOC.ipynb)

Willkommen bei **MINTlab**!  
Dieses Projekt verbindet Mathematik, Informatik, Naturwissenschaften und Technik zu einem innovativen, browserbasierten Kursformat. Die Lernenden durchlaufen eine inspirierende Entdeckungsreise von mathematischen Grundlagen über informatische Werkzeuge bis zu naturwissenschaftlichen Anwendungen und technischer Konstruktion – alles direkt in interaktiven JupyterLab-Notebooks.  
Ziel ist ein moderner, experimenteller MINT-Unterricht mit Fokus auf Visualisierung, Programmierpraxis und projektorientiertem, selbstgesteuertem Lernen.  
Ein Einstieg oder Testen der Materialien ist ohne lokale Installation möglich.

## Projektüberblick

- Interdisziplinärer, projektbasierter Aufbau (Mathematik, Informatik, Naturwissenschaft, Technik)
- Alle Inhalte interaktiv in JupyterLab, plus Visualisierungen und 3D-Modelle
- Für Unterricht, Selbststudium und Open Educational Resources (OER) geeignet

## Schnelleinstieg

Klicke auf den Badge oben, um das Projekt direkt in der Cloud-Umgebung zu starten. Es ist keine lokale Installation erforderlich – mybinder startet automatisch eine frische JupyterLab-Session mit aller benötigten Software[file:23].
